# Data

The simulation datasets used in the paper are not included in this repository.

They consist of HDF5 files containing:

- `spec`   : (N, L) simulated spectra
- `wave`   : (L,) wavelength/velocity grid
- `params` : (N, 9) physical parameters

If you are interested in accessing the data used in the paper, please contact the authors or refer to the journal publication:

https://doi.org/10.3847/1538-4357/ad34b3
